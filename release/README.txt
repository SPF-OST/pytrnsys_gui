To run the pytrnsys GUI double-click on the "pytrnsys-gui.bat" file.

To associate the .pytrnsys file extension with pytrnsys run "install-pytrnsys-gui-RUN-AS-ADMIN.bat"
as administrator. To do that, right click on "install-pytrnsys-gui-RUN-AS-ADMIN.bat" and click on
"Run as administrator". After you've done that, you can click on any file ending in .pytrnsys and
it will automatically be run with pytrnsys.
